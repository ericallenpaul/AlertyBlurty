# v0.1.0 (Initial OSS Preview)

## Highlights

- Zabbix webhook ingestion and event normalization
- Incident grouping by host + trigger
- On-call routing and Twilio SMS notifications
- Self-hosted deployment and operations documentation

## Quality and operations

- CI pipeline for build, format checks, test, and dependency vulnerability scan
- Tagged release workflow that publishes build artifacts and GitHub release notes
- Open source readiness docs: license, contribution guide, and security policy

## Known limitations

- Zabbix integration validation remains dependent on live environment access
- End-to-end QA sign-off is tracked separately
